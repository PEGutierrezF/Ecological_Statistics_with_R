Chapter 3 RStudio Tips, Tricks, and Shortcuts

Shortcuts in Rstudio

Rstudio offers a series of shortcuts to speed up some tasks when you are working in R.

Here are some of the most common and useful shortcuts:


Working on the Code Editor

1- Press Alt GR + - you can get the <-, which is used to designate elements in R.

2- Hide code: press Alt + L.

3- Execute code: press Ctrl+Enter when executing the code line by line.
Ctrl+Alt+R to run the complete code.


Working in dplyr

1- Press ctrl + shift + M you can get the %>% (dplyr: pipe operator), which is used to make the command line continue on the next line.
